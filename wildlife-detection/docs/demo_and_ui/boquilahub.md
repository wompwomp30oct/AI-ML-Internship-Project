## Use BoquilaHUB to run Biodiversity models

BoquilaHUB supports several Biodiversity models natively

![image](https://raw.githubusercontent.com/boquila/boquilahub/main/assets/readme.jpg)

Some of the models include: SpeciesNet, Megadetector, Perch2, Bioclip, many YOLO models, and a lot more!

Automate tasks for camera trap processing, bioacuoustics, embeddings. Connect to real-time cameras, process video and audio in one simple interface.

Repo: https://github.com/boquila/boquilahub/
